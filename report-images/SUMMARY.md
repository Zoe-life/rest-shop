## Insertion Methods

### Method 1: Direct SVG (Word 2016+)
```
Insert → Pictures → This Device → Select SVG file
```

### Method 2: Convert to PNG
1. Open SVG in web browser
2. Take screenshot (high resolution)
3. Insert PNG into Word

### Method 3: Online Converter
- Use CloudConvert or Convertio
- Convert SVG to PNG at 2000px width
- Insert into Word
